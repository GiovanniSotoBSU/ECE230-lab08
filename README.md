## Summary
We learned how multiplexers choose one input from several possible inputs and send that to a single output based on the bit selected. Also how demultiplexers can do the opposite, taking one input and sending that to several outputs. When we have both, we can choose who sends data, and also where it goes.

## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer selects one signal from multiple inputs and sends the selected signal to one output. Select bits are used to decide which input should be passed through.

### In plain English describe the function and use of a demultiplexer.
A demultiplexer takes one input signal and sends that to a specific output. The selected bits decide which output receives the signal and the others that dont. 

### What other uses might these circuits have? (Think Shannon’s)
We can use these circuits to implement Boolean expressions by connecting signals to their inputs.
